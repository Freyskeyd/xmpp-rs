# xmpp-rs

`xmpp-rs` is an XMPP client library (for now). Under active development.

[![Build Status](https://travis-ci.org/Freyskeyd/xmpp-rs.svg?branch=master)](https://travis-ci.org/Freyskeyd/xmpp-rs)

## Install

*Cargo.toml*

```toml
xmpp-rs = "0.1"
```

```
extern crate xmpp;

```
## Build

To build xmpp-rs:

`cargo build --release`
