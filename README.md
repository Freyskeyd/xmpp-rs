# rust-xmpp

`rust-xmpp` is an XMPP client library (for now). Under active development.

[![Build Status](https://travis-ci.org/Freyskeyd/rust-xmpp.svg?branch=master)](https://travis-ci.org/Freyskeyd/rust-xmpp)
## Install

*Cargo.toml*

```toml
rust-xmpp = "0.1"
```

```
extern crate xmpp;

```
## Build

To build rust-xmpp:

`cargo build --release`
